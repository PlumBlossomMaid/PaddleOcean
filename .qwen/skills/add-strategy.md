# paddleOcean Skill — 添加新 Strategy

在 `ocean/strategies/` 下添加新的分布式策略。

## 步骤

1. 继承 `Strategy` 或 `ParallelStrategy`
2. 实现抽象属性：`root_device`、`is_global_zero`
3. 实现 `setup(trainer)` 方法：设备放置 → 精度转换 → 模型包装 → 优化器设置
4. 在 `ocean/strategies/__init__.py` 中注册
5. 在 `ocean/trainer/connectors/__init__.py` 的 `_AcceleratorConnector._resolve_strategy` 中添加映射

## Strategy 关键接口

```python
class MyStrategy(Strategy):
    @property
    def root_device(self) -> Any: ...
    @property
    def is_global_zero(self) -> bool: ...
    def setup(self, trainer: Any) -> None: ...
    def setup_environment(self) -> None: ...
    def model_to_device(self) -> None: ...
    def teardown(self) -> None: ...
    # 分布式通信
    def reduce(self, tensor, reduce_op="mean") -> Any: ...
    def barrier(self, name=None) -> None: ...
    def broadcast(self, obj, src=0) -> Any: ...
    def all_gather(self, tensor, group=None, sync_grads=False) -> Any: ...
```
